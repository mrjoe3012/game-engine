# WHSK Engine
An open source 2D game engine made in c++ for windows.

# Features

- Physics engine.
- Kinematic collision handling.
- Trigger boxes.
- Physics layering.
- Physics culling.
- Physics casting.
- Event handling.
- Primitive shape rendering.
- Render layering.
- Object naming and tagging.
- Sprite rendering.
- Sprite scaling.
- Virtual audio sources.
- Input handling.
- Text rendering
- Text scaling.
- Multiple level support.

# Roadmap

- WHSK Scripting.
- Documentation.
- Level editor.

# Known bugs

- Circle on circle collision detecting in rare cases.
